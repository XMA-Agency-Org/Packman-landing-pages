# /assets folder

Drop these files here before deploying:

- `wordmark-navy.png`        — the Packman wordmark logo (used in nav + footer)
- `logo-icon-navy.png`       — the small icon mark (used inside dashboard mocks)
- `avenir-arabic-book.otf`   — Avenir Arabic Book (weight 400) — for AR body text
- `avenir-arabic-black.otf`  — Avenir Arabic Black (weight 900) — for AR headlines

Until these files exist here, the site will still load — but the wordmark won't show
and Arabic text will fall back to IBM Plex Sans Arabic (still readable, just not the
brand font).
